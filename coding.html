GENERAL INFO.

This page showcases some of my programming and development work. One of my hobbies is coding so I have generated a large list of accomplishments. I am currently employed as a junior web developer and I plan on using my coding ability if I do a course in engineering for University.

DAMN IT DONALD.

This was a game created using JavaScript canvas for our year 10 information technologies class. The game features the player who controls the North Korean Fighter Jet in order to chase down an elusive Donal Trump who is running from "Little Rocket Man" in a red pair of stiletto's. This game was created in collaboration with Benjamin Leembruggen and Tess Styles.

PERSONAL PROJECT.

This was a platformer game I created throughout year 9 for my Personal Project. The goal is to defend yourself from waves of incoming aliens which have a tracking UI. As you progress the waves get harder and harder. I created this game in the program Unity and programmed it using the language C Sharp (C#).
